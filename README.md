# AI Programming with Python Nanodegree

This repository contains the code for the projects developed during the Udacity **AI Programming with Python Nanodegree**. The Nanodegree focuses on providing a strong foundation in Python and core libraries used in AI programming. Below is an overview of the 8 courses and their corresponding lessons.

---

## Overview of Courses

### Course 1: Introduction to AI Programming
- **Welcome to AI Programming**  
  A brief overview of the Nanodegree and AI programming with Python.

---

### Course 2: Introduction to Python for AI Programmers
This course introduces Python programming with an emphasis on libraries and scripts used for solving AI-related problems.

#### Lessons:
1. **Why Python Programming for AI**  
   Overview of Python’s role in AI programming.
2. **Using Python Data Types and Operators in AI Programming**  
   Understanding the building blocks of Python: data types, operators, built-in functions, and type conversion.
3. **Python Data Structures in AI Programming**  
   Using data structures to organize and manipulate different data types.
4. **Using Control Flow in AI Programming**  
   Learn how to control logic in your Python code using loops, conditional statements, and comprehensions.
5. **Using Python Functions in AI Programming**  
   Improve code reusability with functions, variable scope, and lambda expressions.
6. **Python Scripting for AI Programming**  
   Setting up Python programming environments and running scripts.
7. **Introduction to Object-Oriented Python for AI Programming**  
   Learn the basics of object-oriented programming in Python.
8. **Project: Use a Pre-trained Image Classifier to Identify Dog Breeds**  
   A project focused on using Python code and a pre-trained image classifier.

---

### Course 3: Numpy, Pandas, Matplotlib
This course introduces important Python libraries for AI programming: NumPy, Pandas, and Matplotlib.

#### Lessons:
1. **Anaconda**  
   Learn how to use Anaconda for data analysis workflow.
2. **Jupyter Notebooks**  
   Introduction to Jupyter Notebooks for writing and sharing code.
3. **NumPy**  
   Learn the basics of manipulating arrays with NumPy.
4. **Pandas**  
   Working with Pandas Series and DataFrames for data manipulation.
5. **Matplotlib and Seaborn Part 1**  
   Visualization of data using Matplotlib and Seaborn.
6. **Matplotlib and Seaborn Part 2**  
   Further explore data visualizations to represent variable relationships.

---

### Course 4: Linear Algebra Essentials
- Learn the importance of linear algebra in AI and data science.

---

### Course 5: Calculus Essentials
- Covers foundational topics in calculus, including derivatives and the chain rule, essential for neural network training.

---

### Course 6: Neural Networks - AI Programming with Python
An in-depth look at neural networks and how algorithms inspired by the brain can solve AI problems.

#### Lessons:
1. **Welcome to Neural Networks**  
   Introduction to the world of neural networks.
2. **Introduction to Neural Networks**  
   Learn about backpropagation, gradient descent, and implementing neural networks.
3. **Implementing Gradient Descent**  
   Implement gradient descent using matrix multiplication with NumPy.
4. **Training Neural Networks**  
   Explore techniques to improve the training of neural networks.
5. **Deep Learning with PyTorch**  
   Introduction to building deep learning models with PyTorch.

---

### Course 7: Programming Transformer Neural Networks with PyTorch
This course covers the fundamental concepts of Transformers, which are essential for modern AI applications like language models.

#### Lessons:
1. **Introduction to Transformer Neural Networks**  
   Explore the architecture and applications of transformer neural networks.
2. **Building Transformer Neural Networks with PyTorch**  
   Learn to implement transformer models from scratch.
3. **Using Pre-Trained Transformers**  
   Understand how to use pre-trained transformer models for tasks like text generation and Q&A.

---

### Course 8: Create Your Own Image Classifier
In this final project, you'll apply your knowledge to build a state-of-the-art image classification application.

#### Project:
- **Create Your Own Image Classifier**  
   Build an application that trains an image classifier on a dataset and predicts new images.
